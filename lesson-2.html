Введение понятия типа данных

JavaScript

  • numbers (числа)
  • bigint (целые числа)
  • strings (строки)
  • boolean (логический тип данных)
  • null
  • undefined (неприсвоенное значение)
  • object (объект)
  • symbol (cимвол)

Python

  • Numbers (числа)
  • Strings (строки)
  • Lists (списки)
  • Dictionaries (словари)
  • Tuples (кортежи)
  • Sets (множества)
  • Boolean (логический тип данных)
  • None

Java - Примитивы

  • Целые числа - byte, short, int, long
  • Числа с плавающей точкой (иначе вещественные) - float, double
  • Логический - boolean
  • Символьный - char

Java - Ссылочные

  • Все которые создаются с помощью конструкторов
  • Массивы
  • Ссылочная переменная может использоваться в качестве ссылки на любой объект определённого типа данных.
  • Все ссылочные типы имеют значение по умолчанию: null.

Специальные символы в JavaScript
Символ Значение
\b Возврат (Backspace)
\f Перевод или прогон страницы (Form feed)
\n Перевод строки (New line)
\r Возврат каретки (Carriage return)
\t Табуляция (Tab)
\v Вертикальная табуляция (Vertical tab)
\' Апостроф или одинарная кавычка
\" Двойная кавычка
\\ Обратная косая черта (Backslash)
\XXX

Символ в кодировке Latin-1, представленный тремя восьмеричными числами XXX от 0 до 377. Например, \251 (символ ©).
\xXX

Символ в кодировке Latin-1, представленный двумя шестнадцатеричными числами XX от 00 до FF. Например, \xA9 (символ ©).
\uXXXX

Символ в Unicode, представленный четырьмя шестнадцатеричными числами XXXX. Например, \u00A9 (символ ©).
\u{XXXXX} Символ в UTF-32BE. Например, \u{2F804} обозначает то же, что обычная запись \uD87E\uDC04.
